## 115 二维码登录工具
* 简单使用
python 115.py wechatmini
* 可选参数
    * wechatmini： 微信小程序登录 python 115.py wechatmini
    * web： 浏览器登录 python 115.py web
    * android： 安卓登录 python 115.py android
    * ios： ios登录 python 115.py ios
    * linux： linux登录 python 115.py linux
    * mac： mac登录 python 115.py mac
    * windows： windows登录 python 115.py windows
    * tv： tv登录 python 115.py tv
    * alipaymini： 支付宝小程序登录 python 115.py alipaymini
    * qandroid： 115 android登录 python 115.py qandroid
